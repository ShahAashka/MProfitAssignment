## _MProfit Assessment_
https://www.figma.com/file/K9BDUxeCycsLZvYyVmUpPC/MProfit-Mobile-Assignment
### User Flow

• Dashboard - Home Page
• My Spending Page - Home Page --> Click on "Create a Saving goal" card

#### Steps to build

Installing dependencies
```sh
npm install
```
For ios
```sh
cd ios && pod install
```
Executing the application

```sh
npm start
```

Executing on ios

```sh
npx react-native run-ios
```

Executing on android
```sh
npx react-native run-android
```
